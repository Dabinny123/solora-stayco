## Introduction

Solora StayCo is a curated hotel booking and hosting platform that focuses on aesthetic, minimalist accommodations. This manual will guide you through using all features of the platform.

---

## Getting Started

### Creating an Account

1. **Visit the Homepage**
   - Navigate to the Solora StayCo website
   - Click "Sign Up" in the top right corner

2. **Choose Your Role**
   - **Guest**: Select "Book accommodations (Guest)" if you want to book stays
   - **Host**: Select "List my property (Host)" if you want to list your property

3. **Fill in Your Information**
   - Enter your full name
   - Enter your email address
   - Choose a secure password (minimum 6 characters)
   - Confirm your password

4. **Complete Registration**
   - Click "Sign Up"
   - Check your email for verification (if required)
   - You'll be redirected to your dashboard

### Signing In

1. Click "Sign In" in the top right corner
2. Enter your email and password
3. Click "Sign In"
4. You'll be redirected to your role-specific dashboard

### Forgot Password

1. Click "Sign In"
2. Click "Forgot password?"
3. Enter your email address
4. Check your email for password reset instructions

## Guest Features

### Exploring Listings

1. **Browse All Listings**
   - Click "Explore" in the navigation menu
   - Browse through available accommodations
   - Use filters to narrow down your search

2. **Search and Filter**
   - **Search Bar**: Search by location, title, or description
   - **Category**: Filter by Home, Experience, or Service
   - **City**: Filter by specific city
   - **Price Range**: Set minimum and maximum price
   - **Guests**: Filter by number of guests

3. **View Featured Listings**
   - Click "Featured" tab to see highly-rated listings
   - Featured listings have ratings of 4+ stars

### Viewing Listing Details

1. Click on any listing card to view details
2. **Listing Information Includes**:
   - Photo gallery
   - Full description
   - Location and address
   - Amenities
   - Capacity (guests, bedrooms, beds, bathrooms)
   - Reviews and ratings
   - Pricing information

3. **Save to Wishlist**
   - Click the heart icon in the top right
   - Sign in if prompted
   - View saved listings in your Wishlist

### Making a Booking

1. **Select Dates and Guests**
   - On the listing detail page, select check-in and check-out dates
   - Choose number of guests
   - Review the price breakdown

2. **Click "Reserve" or "Sign in to Reserve"**
   - If not signed in, you'll be redirected to sign in
   - If signed in, you'll proceed to the booking page

3. **Complete Booking Form**
   - Verify trip details (dates, guests)
   - Enter guest information:
     - Full name
     - Email
     - Phone number (optional)
     - Special requests (optional)

4. **Choose Payment Method**
   - **E-Wallet**: Pay using your wallet balance
   - **Credit/Debit Card**: Pay with card (coming soon)

5. **Review and Confirm**
   - Review booking summary
   - Check total amount
   - Click "Confirm Booking"

6. **Booking Confirmation**
   - You'll see a confirmation page
   - Booking details will be sent to your email
   - View booking in "My Bookings"

### Managing Bookings

1. **View Bookings**
   - Go to Guest Dashboard
   - Click "My Bookings"
   - View all your bookings with status

2. **Booking Statuses**:
   - **Pending**: Awaiting host confirmation
   - **Confirmed**: Booking confirmed by host
   - **Completed**: Stay has ended
   - **Cancelled**: Booking was cancelled

### Wishlist

1. **Add to Wishlist**
   - Click heart icon on any listing
   - Listing is saved to your wishlist

2. **View Wishlist**
   - Go to Guest Dashboard
   - Click "Wishlist"
   - View all saved listings
   - Click on any listing to view details

3. **Remove from Wishlist**
   - Go to listing detail page
   - Click heart icon again to remove

### Account Settings

1. **Access Settings**
   - Click on your profile icon in the header
   - Select "Settings"

2. **Update Profile**
   - Upload profile photo
   - Update display name
   - Update phone number
   - View account information

## Host Features

### Creating a Listing

1. **Access Create Listing**
   - Go to Host Dashboard
   - Click "Create New Listing"

2. **Fill in Basic Information**
   - **Title**: Enter a descriptive title
   - **Description**: Write a detailed description
   - **Category**: Choose Home, Experience, or Service
   - **Type**: Specify type (e.g., Apartment, Villa, Workshop)

3. **Add Location**
   - Enter full address
   - City, State, Country
   - ZIP code

4. **Set Pricing**
   - Base price per night
   - Cleaning fee (optional)
   - Security deposit (optional)
   - Currency

5. **Specify Capacity**
   - Maximum guests
   - Number of bedrooms
   - Number of beds
   - Number of bathrooms

6. **Add Amenities**
   - Type amenity name and click "Add"
   - Examples: WiFi, Pool, Parking, Air Conditioning

7. **Upload Photos**
   - Click "Upload Photos"
   - Select multiple images
   - Set featured photo (first photo is default)
   - Remove photos if needed

8. **Publish or Save as Draft**
   - **Publish Listing**: Makes listing active and visible
   - **Save as Draft**: Saves for later editing

### Managing Listings

1. **View All Listings**
   - Go to Host Dashboard
   - Click "Manage Listings" or navigate to "My Listings"
   - Filter by status: All, Active, Draft, Inactive

2. **Edit Listing**
   - Click "Edit" on any listing
   - Update information
   - Save changes

3. **Delete Listing**
   - Click "Delete" on any listing
   - Confirm deletion
   - Listing will be permanently removed

### Managing Bookings

1. **View Bookings**
   - Go to Host Dashboard
   - Click "View Bookings"
   - Filter by status: All, Pending, Confirmed, Completed, Cancelled

2. **Today's Check-ins**
   - View guests checking in today
   - Prepare for guest arrival

3. **Upcoming Bookings**
   - View all future bookings
   - Plan ahead for guest stays

4. **Confirm Booking**
   - For pending bookings, click "Confirm"
   - Booking status changes to "Confirmed"
   - Guest receives confirmation

5. **Cancel Booking**
   - Click "Cancel" on any booking
   - Provide cancellation reason
   - Booking status changes to "Cancelled"

### Messages

1. **Access Messages**
   - Go to Host Dashboard
   - Click "Messages"
   - View all conversations

2. **View Conversation**
   - Click on any conversation
   - View message history
   - Messages are automatically marked as read

3. **Send Message**
   - Type your message
   - Click "Send"
   - Guest receives notification

## Admin Features

### Dashboard Overview

1. **View Analytics**
   - Total bookings
   - Active listings
   - Total users
   - Total revenue
   - Pending payments

2. **Review Management**
   - View best reviews (highest ratings)
   - View lowest reviews (lowest ratings)
   - Monitor platform quality

### User Management

1. **Access User Management**
   - Go to Admin Dashboard
   - Click "User Management"

2. **View All Users**
   - See all registered users
   - Filter by role (Guest, Host, Admin)
   - Search by name, email, or ID

3. **Change User Roles**
   - Select new role from dropdown
   - Confirm role change
   - User permissions update immediately

### Payment Review

1. **Access Payment Review**
   - Go to Admin Dashboard
   - Click "Payment Review"

2. **View Payments**
   - See all payment transactions
   - Filter by status: All, Pending, Processing, Completed, Failed, Refunded

3. **Manage Payments**
   - **Confirm Payment**: Mark pending payments as completed
   - **Mark as Failed**: Mark failed payments
   - **Process Refund**: Issue refunds for completed payments

### Service Fee Control

1. **Access Service Fee**
   - Go to Admin Dashboard
   - Click "Service Fee"

2. **Configure Service Fee**
   - Enter percentage (e.g., 10 for 10%)
   - View example calculation
   - Click "Save Changes"
   - Changes apply to new bookings only

### Reports Generation

1. **Access Reports**
   - Go to Admin Dashboard
   - Click "Reports"

2. **Generate Report**
   - Select report type:
     - Bookings Report
     - Payments Report
     - Users Report
     - Listings Report
   - Optionally set date range
   - Click "Generate Report"

3. **View Report Data**
   - See total counts
   - View breakdowns by status/role/category
   - Review detailed information

## E-Wallet & Payments

### E-Wallet Management

1. **Access E-Wallet**
   - Go to Guest Dashboard
   - Click "E-Wallet"

2. **View Balance**
   - See current wallet balance
   - View transaction history

3. **Add Funds**
   - Enter amount or use quick buttons ($50, $100, $200)
   - Click "Add Funds"
   - Funds are added instantly

4. **Transaction History**
   - View all deposits and withdrawals
   - See transaction dates and amounts
   - Check balance after each transaction

### Making Payments

1. **During Booking**
   - Select "E-Wallet" as payment method
   - View current balance
   - If balance is sufficient, payment processes automatically
   - If insufficient, add funds before completing booking

2. **Payment Processing**
   - E-wallet payments are instant
   - Balance is deducted immediately
   - Payment confirmation is sent

### Points & Rewards

1. **Earning Points**
   - **Bookings**: 1 point per $10 spent
   - **Reviews**: 10 points for submitting a review
   - **First Booking**: 50 bonus points

2. **Viewing Points**
   - Points are displayed in your account
   - Track your points balance

3. **Redeeming Points**
   - 100 points = $1 discount
   - Redeem during checkout (coming soon)

## Troubleshooting

### Common Issues

**Can't Sign In**
- Verify email and password are correct
- Use "Forgot Password" to reset
- Check if email is verified

**Payment Failed**
- Check wallet balance
- Ensure sufficient funds
- Try adding funds first

**Listing Not Showing**
- Verify listing status is "Active"
- Check if listing is published (not draft)
- Contact support if issue persists

**Booking Not Confirmed**
- Wait for host confirmation
- Check booking status in "My Bookings"
- Contact host via Messages

**Can't Upload Photos**
- Check file size (max 5MB)
- Ensure file is an image format (JPG, PNG, GIF)
- Try refreshing the page

### Getting Help

- **Contact Support**: Use the contact form on the website
- **Check FAQ**: Visit the FAQ section
- **Email Support**: support@solora-stayco.com

## Mood-Based Booking System

### Guests
- Use the **Mood-Based Stays** carousel on the Explore page
- Selecting a mood filters listings instantly and stores your preference
- Notifications confirm booking submissions and approvals

### Hosts
- In the listing form, fill in the **Mood & Ambience** section:
  - Assign mood tags
  - Add ambience tags, color palettes, and lighting styles
  - Set an aesthetic score (0–100)
  - Write mood highlights for marketing copy

### Admins
- Navigate to **Admin Dashboard → Mood Library**
- Create, activate/deactivate, or delete moods
- Mood updates take effect immediately across the platform
